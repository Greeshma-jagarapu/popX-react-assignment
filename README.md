# popX-react-assignment
A modern, responsive user onboarding and profile management interface built with **React** and styled for seamless mobile-first experiences. This application includes multi-screen transitions, form validations, and user profile state management.

## Features
* **Welcome Page** - A welcome page having signup and login buttons
* **Signup Page** - A page for taking user details to create an user account
* **SignIn Page** - A page which allows user to access their account if they have already created an account by verifying their email and password
* **Account Settings Page** - A page which shows user account

## Tech Stack
* Reactjs with vite
* Tailwindcss

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### 📋 Prerequisites
Before you begin, ensure you have the following installed on your computer:
* **Web Browser:** Google Chrome, Mozilla Firefox, or Microsoft Edge.
* **Code Editor:** Visual Studio Code (recommended) or any text editor.
* **Nodejs:** To make react work
* **Git:** Installed on your local machine to clone the files.

### ⚙️ Installation & Setup
Follow this quick terminal sequence to get your local environment running:

**Clone the repository, navigate and launch:**
 Download the project files securely from GitHub to your desktop.
 git clone <your-repo-url>
 cd <your-cloned-repo-folder>
 npm install
 npm run dev

 **I have used localStorage to store user information just to show a demo**
